🍑 Fruits du Démon
CRUD complet — /fruits

Lister les fruits
Récupère tous les fruits, filtrable par type
GET
GET /fruits?type={type}

🔍
Rechercher un fruit
Recherche par nom (regex insensible à la casse)
GET
GET /fruits/search?name={name}

POST /fruits · Body: JSON
✏️

🗑️
Supprimer un fruit
Supprime définitivement un fruit via son ID
